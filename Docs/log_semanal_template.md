# Log de Estudos — Python aplicado à Engenharia

Este arquivo deve ser preenchido ao final de cada semana de estudo. Manter no repositório 
GitHub, na pasta `registros/`. O preenchimento deve levar menos de cinco minutos. Uma vez 
por mês, anexar este arquivo à conversa com o Claude (dentro do Project) para análise de 
evolução.

Instruções de uso: copie o bloco "Modelo de entrada semanal" abaixo e cole no início do 
arquivo a cada nova semana, mantendo as semanas anteriores logo abaixo, em ordem cronológica 
inversa (a mais recente primeiro).
